# Astrobound

Browser 3D space exploration: fly through a procedural solar system, land on planets, and walk their surfaces.

**Stack:** TypeScript · three.js · miniplex · Rapier · Vite

## Run

```bash
npm install
npm run dev
```

Dev server defaults to Vite's port; for the usual local port:

```bash
npx vite --port=4000
```

Typecheck:

```bash
npx tsc --noEmit
```
